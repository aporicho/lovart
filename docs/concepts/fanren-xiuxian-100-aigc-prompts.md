# 凡人修仙传 100 张概念设计 AIGC 提示词文档

## 目的

这份文档由 `AIGC提示词设计师.md` 的无损转换规则生成，承接 `fanren-xiuxian-100-concept-designs.md`。

它的目标不是重新设计概念，也不是把结构字段压缩成一句漂亮描述，而是把每个概念的 9 个源字段完整转换成模型可执行的 prompt：`核心`、`总画面描述`、`叙事瞬间`、`主体`、`场景`、`镜头`、`构图`、`暗示`、`画风`。

## 本次对比结论

上一版 prompt 文档的主要问题是压缩过度：

- `总画面描述` 被当成主要来源，部分条目的 `核心` 精确命题被弱化。
- `叙事瞬间` 有时被融入散文句子，失去独立可检查性。
- `场景` 中的具体物件容易被删减，例如竹牌、阵旗、旧血痕、收费石碑、破符、残碑等。
- `暗示` 经常被概括成危险、压迫、悬疑，丢失画外故事。
- `画风` 的单图差异容易被项目级风格覆盖，例如道具写实设定、政治电影风、生物恐怖概念、寒地史诗环境等。
- 单段 prompt 难以自动审查字段覆盖，不利于 agent 批量生成。

因此本版改为字段无损 prompt。每条 prompt 都保留源字段标签，方便模型理解，也方便 agent 检查和后续压缩。

## 使用方式

推荐拼装方式：

```text
最终 prompt = 项目级 Prompt + 单图无损 Prompt + 统一负向约束
```

模型参数不要写进 prompt。尺寸、比例、质量、数量、fast / relax / auto、输出格式必须通过 Lovart 工具查询合法值：

```bash
lovart setup
lovart plan --intent image-concept
lovart config <model>
lovart quote <model> --body-file request.json
lovart generate <model> --body-file request.json --dry-run
```

## 项目级 Prompt

```text
电影级写实东方玄幻概念设计，凡人流修仙气质，低饱和真实材质，克制法术光效，古代中国建筑、器物、衣料和生活痕迹为基础，少量水墨气韵，厚涂概念笔触，强叙事性画面，强调凡人与仙道之间的阶层差、危险、孤独、代价和长生压迫。画面要像影视概念设计 keyframe 或生产设定图，而不是普通插画。
```

## 统一负向约束

```text
避免二次元偶像脸、现代建筑、现代服装、科幻机甲、西幻盔甲、廉价页游特效、过度发光、满屏粒子、塑料材质、游戏 UI、文字、水印、logo、无叙事站桩、过度华丽导致凡人感消失。
```

## 批量一致性规则

- 统一世界观：凡人流修仙，写实东方玄幻，古代中国器物基础。
- 统一审美：低饱和、真实材质、克制法术光效、电影 keyframe 感。
- 统一叙事：每张图必须发生在具体一秒，而不是抽象设定展示。
- 字段无损：每条单图 prompt 必须保留 9 个源字段。
- 差异化重点：主体、场景、镜头、构图、光线、事件、暗示物必须变化。
- 参数独立：prompt 不声明尺寸、比例、质量、数量，交给 Lovart request body。

### 001. 凡人少年初入仙途

```text
生成一张电影级写实东方玄幻概念设计图。
核心叙事：凡人与仙门的第一次距离。
完整画面：泥路从村口蜿蜒伸向远山，背竹篓的少年停在路中央，脚边旧布包还沾着家里的尘土；炊烟在他身后低低散开，云层深处却隐约露出巨大的仙门轮廓，山门像压在天上的阴影。广角低机位让少年显得很小，观众能同时感到乡土的温度和仙途尺度的压迫。
叙事瞬间：少年刚离开村口，第一次看见云中山门。
第一主体：背竹篓的少年。
场景世界观：泥路、炊烟、远山、云中若隐若现的仙门。
镜头语言：24mm 广角远景，低机位轻微仰拍，深景深。
构图语言：尺度对比，少年很小，山门压在画面上方。
暗示叙事：脚边旧布包说明他刚告别家人。
画风材质：电影写实东方玄幻，低饱和乡土色，仙门光效克制。
```

### 100. 大道无声

```text
生成一张电影级写实东方玄幻概念设计图。
```
